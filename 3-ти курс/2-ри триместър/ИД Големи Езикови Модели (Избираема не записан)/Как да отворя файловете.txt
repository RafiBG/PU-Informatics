Може да отвориш файловете ipynb чрез Visual Studio Code

Възможно е автоматично Visual Studio Code да го отвори в Restricted mode
което значи че няма да видиш нищо. Трябва да видиш една синя лента отгоре
и да натистнеш Manage и избери отляво Trusted Window и вече ще виждаш файла както
би трябвало да бъде.
